# Deploy Institutional Quality v2 Patch

1. Upload `MOIL_InstitutionalQualityV2_Patch.zip` to `/workspaces/MOIL`.
2. Run the deployment block provided by ChatGPT.
3. Run `python institutional_tracker.py` to refresh institutional cache.
4. Start Streamlit.
5. Open the Institutional sync and Data quality tabs.

Do not commit `.streamlit/secrets.toml` or zip files.
